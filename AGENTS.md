# PaperPiggy 教程仓库维护规则

本仓库只存放面向普通用户的公开教程、示例和截图，不存放 PaperPiggy 程序代码、真实知识库、索引、API Key、私人文献或本机绝对路径。

## 写作规则

1. 默认使用简体中文。
2. 面向第一次使用的普通研究者，先写操作结果，再解释原理。
3. 每个操作章节尽量包含：目标、操作步骤、成功标志、常见问题。
4. 不在正文写死“当前版本号”“当前工具数量”等容易漂移的事实；需要时链接到最新版和程序内真实界面。
5. 本地模型体积写“约 1GB，以当前界面提示为准”，不要复制多个精确数字。
6. 不笼统宣称“全程离线”：本地模式可离线检索；API 检索和大模型对话会向用户选择的服务商发送必要文本。
7. 所有截图必须隐藏 API Key、用户名、本机绝对路径、私人收藏夹、未公开文献和研究项目。
8. 应用页面、按钮、流程发生变化时，同步检查相关章节。
9. 发布前通读面向用户的正文，删除过期说法和修订残留。
10. 软件问题统一引导到主仓库，不在教程仓库另建一套产品问题入口。
