# PaperPiggy 新手教程

这是 [PaperPiggy（论文小猪）](https://github.com/DrinkTea905/paper-piggy) 的独立中文使用指南。

- 教程网站：https://drinktea905.github.io/paper-piggy-guide/
- 下载最新版：https://github.com/DrinkTea905/paper-piggy/releases/latest
- 程序源码与问题反馈：https://github.com/DrinkTea905/paper-piggy

教程面向普通用户，内容包括安装、首次建库、深索、全文检索、内置对话、综述库、Agent 接入、备份与故障排查。

## 如何修改

教程正文位于 `docs/`，每一章是一个 Markdown 文件。可以直接在 GitHub 网页中打开文件，点击右上角铅笔图标编辑并提交。修改 `main` 分支后，GitHub Pages 会自动重新发布。

维护规则见 [AGENTS.md](AGENTS.md)。
